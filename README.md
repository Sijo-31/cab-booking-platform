# Cab Booking Platform with Real-Time Driver Tracking

This project is a simplified ride-hailing backend similar to Uber/Ola.

Features:
- Ride request system
- Nearest driver assignment using Redis GEO
- Real-time driver tracking using WebSockets
- Interactive map with React Leaflet
- Distance and price estimation

Tech Stack:
Backend: FastAPI, Python  
Frontend: Next.js, React  
Database: PostgreSQL  
Realtime: WebSockets  
Location Search: Redis GEO  
Containerization: Docker
